### On-Page SEO Checklist

**Homepage Optimization**
- [ ] Title: "Lume - Open Source CRM & Database Builder (Self-Hosted)"
- [ ] Meta description: "Build custom CRM, databases & automations without code. Self-hosted, REST API, 23 modules. Open source alternative to Airtable."
- [ ] H1: "The Open Source CRM & Database Platform for Teams"
- [ ] Internal links: 15+ to key pages (features, docs, pricing, use cases)
- [ ] Schema.org markup: Organization + SoftwareApplication
- [ ] Open Graph tags: og:title, og:description, og:image, og:url
- [ ] Word count: 2,500-3,000 words
- [ ] Images: 3-4 high-quality product screenshots
- [ ] CTA placement: Newsletter signup, demo request, self-hosted guide
- [ ] Page speed: Target < 2 seconds (Lighthouse > 90)

**Key Landing Pages**
```
/use-cases/
├─ CRM (title: "Open Source CRM Solution | Lume")
├─ Project Management (title: "Project Management Tool | Lume")
├─ Inventory Management (title: "Inventory Management Software | Lume")
├─ Team Collaboration (title: "Team Collaboration Platform | Lume")
└─ Custom Database (title: "No-Code Database Builder | Lume")

/comparison/
├─ Lume vs Airtable (target: #1)
├─ Lume vs Notion (target: #2-3)
├─ Lume vs Salesforce (target: #3-5)
├─ Lume vs Monday.com (target: #2-3)
└─ Open Source Comparison Table

/deployment/
├─ Docker deployment guide (target: "docker self-hosted app")
├─ AWS deployment guide (target: "deploy app on AWS")
├─ Kubernetes setup (target: "kubernetes database app")
└─ VPS deployment (target: "self-hosted Linux VPS")
```

**Technical SEO**
- [ ] XML sitemap: `/sitemap.xml` (auto-generated, updated weekly)
- [ ] robots.txt: Optimized for crawler efficiency
- [ ] Canonical URLs: Set on all pages (avoid duplicate content)
- [ ] Structured data (JSON-LD):
  - [ ] Organization schema
  - [ ] SoftwareApplication schema
  - [ ] FAQPage schema (for FAQ page)
  - [ ] BreadcrumbList schema (navigation hierarchy)
- [ ] Mobile responsiveness: Tested on iPhone 12, iPad, Android
- [ ] Core Web Vitals:
  - [ ] LCP (Largest Contentful Paint): < 2.5s
  - [ ] FID (First Input Delay): < 100ms
  - [ ] CLS (Cumulative Layout Shift): < 0.1
- [ ] SSL/HTTPS: Force HTTPS, HTTP/2 enabled
- [ ] Crawlability: All important pages linked internally
