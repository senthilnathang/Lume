## Email Template Specifications

### Design
- Single-column responsive layout
- Brand blue accent color (#0066FF)
- Logo in header
- Social icons in footer
- CTA button in brand color

### Best Practices
- Subject line: 50 characters max
- Preview text: Use first email sentence
- Personalization: Use {{first_name}} if available
- Links: All use Google Analytics UTM parameters
  - utm_source=email
  - utm_medium=launch_sequence
  - utm_campaign=september
- Mobile optimized: Test on iOS Mail, Gmail, Android

### Send Schedule
- Pre-launch emails: Send 3x before launch (14, 7, 1 day before)
- Launch day: Send 2x on launch day (live announcement + tour)
- Post-launch: Send 2x per week for first 4 weeks
- Then: 1x per week (ongoing newsletter)

### Metrics to Track
- Open rate (target: 30%)
- Click rate (target: 5%)
- Unsubscribe rate (target: <0.5%)
- Conversion to free trial (target: 3%)

---

## List Management

### Segmentation

**Segment 1: "Early Access"**
- Opted in pre-launch
- Send all emails
- Highest engagement expected

**Segment 2: "Blog Subscribers"**
- Signed up via blog
- Send blog-focused content
- Skip some commercial emails

**Segment 3: "Social Followers"**
- Clicked from social
- Send community highlights
- Less technical content

### Unsubscribe Handling
- One-click unsubscribe (required by law)
- Offer to reduce frequency instead
- Exit survey: "What didn't interest you?"

## Email Tools

**Recommended platform:** Mailchimp (free tier) or Substack (newsletter focused)

**Setup:**
1. Import emails from landing page signups
2. Create segments based on source
3. Schedule drip campaigns
4. Set up automation (welcome series)
5. Monitor opens, clicks, unsubscribes

**Personalization:**
- Use {{first_name}} from signup
- Reference source ("You signed up from Twitter")
- Dynamic content blocks based on segment
