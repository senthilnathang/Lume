Key Design Principles

  • Single Entry Point: All operations (entity CRUD, workflow trigger, view render) go through Runtime.execute()
  • Unified Pipeline: Permissions, validation, hooks, execution, events all flow through one pipeline
  • Metadata-Driven: Schema, permissions, views, workflows all defined in metadata registry
  • Async Foundation: Event bus enables workflows, agents, and real-time updates without blocking
  • ORM Agnostic: Pipeline selects right adapter (Prisma or Drizzle) based on entity definition
